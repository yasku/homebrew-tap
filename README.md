# yasku/homebrew-tap

Homebrew tap for [yasku](http://agustiny-dev.ar)'s tools.

## Usage

```bash
brew tap yasku/tap
brew install <formula>
```

Or install directly:

```bash
brew install yasku/tap/<formula>
```

## Available formulae

| Formula | Description |
|:--|:--|
| [`gone`](https://github.com/yasku/gone) | macOS uninstaller and system monitor TUI |
